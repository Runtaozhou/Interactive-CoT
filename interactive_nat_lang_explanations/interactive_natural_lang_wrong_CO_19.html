Problem Statement

Jason sent 220 text messages on Monday, half as many text messages on Tuesday, and 50 text messages each day Wednesday through Friday. How many text messages did he send on average during those five days?

Problem Understanding

Monday text messages: 220

Tuesday text messages: half of Monday's

Wednesday-Friday text messages: 50 per day

Total days: 5

What we need to find

The average number of text messages Jason sent during the five days.

Step-by-Step Explanation
First find how many text messages he sent on Tuesday:
Monday messages ÷ 2
= 220 messages / 2 = 110 messages
Then find how many text messages he sent Wednesday-Friday:
daily messages × number of days
= 50 messages/day * 2 days = 100 messages
Then add up all the messages he sent to find the total:
Monday messages + Tuesday messages + Wednesday-Friday messages
= 220 messages + 110 messages + 100 messages = 430 messages
Then divide that number by the number of days to find the average number of messages he sent per day:
total messages ÷ number of days
= 430 messages / 5 days = 86 messages/day
Final Answer: 86
