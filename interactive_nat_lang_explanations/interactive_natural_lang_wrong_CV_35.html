Problem Statement

A family of 6 (2 adults and 4 kids) are to divide a watermelon such that each adult gets a slice that is twice as big as that of each kid. What percentage of the watermelon does each adult get?

Problem Understanding

Family composition: 2 adults and 4 kids (total 6 people)

Adult slice size: 2 times kid slice size

What we need to find

The percentage of the watermelon that each adult gets.

Step-by-Step Explanation
Define the percentage of watermelon that each adult gets: x : percentage each adult gets
Calculate the percentage each kid gets based on the adult's portion:
kid's percentage = adult's percentage / 2
= x/2 = x/2
Express the total watermelon as the sum of all portions:
total = 2 adults' portions + 4 kids' portions
= 2*x + 4*(x/2) = 2x + 4*(x/2)
Set up an equation since the total must equal 100%:
2x + 4x = 100
= 2x + 4x = 100 = 2x + 4x = 100
Solve for x to find the percentage each adult gets:
x = 100/6
= x = 100/6 = 16.67%
Final Answer: 16.67%
