Problem Statement

Pablo made 4 stacks of toy blocks. The first stack was 5 blocks tall. The second stack was 2 blocks taller than the first. The third stack was 5 blocks shorter than the second stack, and the last stack was 5 blocks taller than the third stack. How many toy blocks did Pablo use in all?

Problem Understanding

First stack height: 5 blocks

Second stack: 2 blocks taller than first

Third stack: 5 blocks shorter than second

Last stack: 5 blocks taller than third

What we need to find

The total number of toy blocks Pablo used in all stacks.

Step-by-Step Explanation
Calculate the height of the second stack:
first_stack_height + 2
= 5 + 2 = 7 blocks
Calculate the height of the third stack:
second_stack_height - 5
= 7 - 5 = 2 blocks
Calculate the height of the last stack:
third_stack_height + 5
= 2 + 5 = 7 blocks
Add up all blocks to find the total:
first_stack_height + second_stack_height + third_stack_height + last_stack_height
= 5 + 7 + 3 + 7 = 22 blocks
Final Answer: 22
