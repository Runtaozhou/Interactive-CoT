Problem Statement

Together Lily, David, and Bodhi collected 43 insects. Lily found 7 more than David. David found half of what Bodhi found. How many insects did Lily find?

Problem Understanding

Total insects collected: 43

Lily found 7 more insects than David

David found half of Bodhi's insects

What we need to find

The number of insects Lily found.

Step-by-Step Explanation
Define B as the number of insects Bodhi collected: B : number of insects Bodhi collected
Express David's insects in terms of B:
David = B/2
= B/2 insects
Express Lily's insects in terms of David's (and thus B):
Lily = B/2 + 7
= B/2 + 7 insects
Set up equation using the total number of insects:
B + B/2 + 7 = 43
Simplify the equation:
2B + B = 36
Solve for B:
3B = 36
= B = 12 insects
Calculate how many insects David found:
David = B/2
= 12/2 = 6 insects
Calculate how many insects Lily found:
Lily = David + 7
= 6 + 7 = 13 insects
Final Answer: Lily found 13 insects
