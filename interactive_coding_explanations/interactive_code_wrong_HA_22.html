Problem Statement

John decides to start collecting art. He pays the same price for his first 3 pieces of art and the total price came to $45,000. The next piece of art was 50% more expensive than those.

Problem Summary
  • Number of initial art pieces: 3, total cost: $45,000
  • Next piece: 50% more expensive than initial pieces

What we need to find

We need to calculate the total amount John spent on all art pieces.

Python Solution
1 # Calculate the cost of each initial art piece
2 cost_per_piece = 45000 / 3
3 # Calculate the additional cost for the next piece
4 additional_cost = cost_per_piece * 0.5
5 # Calculate the cost of the fourth piece
6 fourth_piece_cost = cost_per_piece + additional_cost
7 # Calculate the total cost including delivery fee
8 total_cost = 45000 + fourth_piece_cost + 2500
