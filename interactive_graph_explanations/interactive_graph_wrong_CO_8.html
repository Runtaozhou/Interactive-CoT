Problem Statement

Scott runs 3 miles every Monday through Wednesday. Then he runs twice as far he ran on Monday every Thursday and Friday. How many miles will he run in a month with 4 weeks?

Problem Summary
  • Miles run Monday through Wednesday: 3 miles per day
  • Miles run Thursday and Friday: twice Monday's miles
  • Number of weeks in the month: 4 weeks
Goal: Calculate the total miles Scott will run in a month
