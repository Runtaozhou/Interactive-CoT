Problem Statement:

Suzanne wants to raise money for charity by running a 5-kilometer race. Her parents have pledged to donate $10 for her first kilometer and double the donation for every successive kilometer. If Suzanne finishes the race, how much money will her parents donate?

Step By Step Explanation:

Step 1: Calculate the donation for the 2nd kilometer.

Since the donation doubles for each successive kilometer, we multiply the first kilometer donation by 2.

$10 × 2 = $20

So the donation for the 2nd kilometer is $20.

Step 2: Calculate the donation for the 3rd kilometer.

We multiply the second kilometer donation by 2.

$20 × 2 = $40

So the donation for the 3rd kilometer is $40.

Step 3: Calculate the donation for the 4th kilometer.

We multiply the third kilometer donation by 2.

$40 × 2 = $80

So the donation for the 4th kilometer is $80.

Step 4: Calculate the donation for the 5th kilometer.

We multiply the fourth kilometer donation by 2.

$80 × 2 = $160

So the donation for the 5th kilometer is $160.

Step 5: Sum all donations to find the total amount.

We add up the donations for all five kilometers:

$10 + $20 + $40 + $80 + $160 = $310

So the total donation will be $310.

Final Answer:

310
