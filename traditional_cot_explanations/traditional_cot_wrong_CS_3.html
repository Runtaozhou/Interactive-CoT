Pablo made 4 stacks of toy blocks. The first stack was 5 blocks tall. The second stack was 2 blocks taller than the first. The third stack was 5 blocks shorter than the second stack, and the last stack was 5 blocks taller than the third stack. How many toy blocks did Pablo use in all?

The second stack is 2 blocks taller than the first stack.

First stack height = 5 blocks

Second stack height = 5 + 2 = 7 blocks

Step 2: Calculate the height of the third stack.

The third stack is 5 blocks shorter than the second stack.

Second stack height = 7 blocks

Third stack height = 7 - 5 = 2 blocks

Step 3: Calculate the height of the last stack.

The last stack is 5 blocks taller than the third stack.

Third stack height = 2 blocks

Last stack height = 2 + 5 = 7 blocks

Step 4: Add up all blocks to find the total.
