Problem Statement:

Jackson is laying tile in a courtyard that measures 10 feet by 25 feet. He needs 4 tiles per square foot of space. 40% of the tiles are green marble that costs $3/tile, and the rest are red tile that costs $1.50/tile. How much does he pay total for tile?

Step By Step Explanation:

Step 1: First find the area of the courtyard.

Using the formula: length + width

10 feet + 25 feet = 35 feet

Step 2: Then multiply the area by the number of tiles per square foot to find the total number of tiles needed.

Using the formula: courtyard_area × tiles per square foot

35 feet * 4 tiles/square foot = 140 tiles

Step 3: Then multiply the number of tiles by the percentage that are green to find the total number of green tiles.

Using the formula: total_tiles × percentage of green tiles

140 tiles * 40% = 56 tiles

Step 4: Subtract that number from the total number of tiles to find the number of red tiles.

Using the formula: total_tiles - green_tiles

140 tiles - 56 tiles = 84 tiles

Step 5: Multiply the number of red tiles by the price per red tile to find the cost of the red tiles.

Using the formula: red_tiles × price per red tile

84 tiles * $1.50/tile = $126

Step 6: Multiply the number of green tiles by the price per green tile to find the cost of the green tiles.

Using the formula: green_tiles × price per green tile

56 tiles * $3.00/tile = $168

Step 7: Then add the cost of each type of tile to find the total cost.

Using the formula: red_tiles_cost + green_tiles_cost

$126 + $168 = $294

Final Answer:

294
