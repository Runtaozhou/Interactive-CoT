Problem Statement:

A family of 6 (2 adults and 4 kids) are to divide a watermelon such that each adult gets a slice that is twice as big as that of each kid. What percentage of the watermelon does each adult get?

Step By Step Explanation:

Step 1: Let's define the percentage of watermelon that each adult gets as x.
Step 2: Since each adult gets a slice that is twice as big as each kid's slice, we can calculate that each kid gets x/2 percent of the watermelon.
Step 3: Now we can express the total watermelon as the sum of all portions. We have 2 adults each getting x percent, and 4 kids each getting x/2 percent.
Total percentage = 2x + 4(x/2) = 2x + 2x = 4x
Step 4: Since the total watermelon represents 100%, we can set up the equation:
2x + 4x = 100
Step 5: Now we can solve for x to find the percentage each adult gets:
x = 100/6 = 16.67%

Final Answer:

16.67
