Problem Statement:

The size of a bathroom is 96 sq ft. If the width of the bathroom is 8 feet and the owner wants to extend it by 2 feet on each side, what is the new area of the bathroom?

Step By Step Explanation:

Step 1: Calculate the original length of the bathroom using the area and width.

We know the area is 96 sq ft and the width is 8 ft.

Using the formula: length = area ÷ width

length = 96 sq ft ÷ 8 ft = 12 ft

So the original length of the bathroom is 12 ft.

Step 2: Calculate the new length by adding the extensions to both sides.

The owner wants to extend by 2 feet on each side.

New length = original length + extension on one side + extension on other side

New length = 12 ft + 2 ft + 2 ft = 16 ft

Step 3: Calculate the new width by adding the extensions to both sides.

Original width is 8 ft.

New width = original width + extension on one side + extension on other side

New width = 8 ft + 2 ft + 2 ft = 12 ft

Step 4: Calculate the new area by multiplying the new length and new width.

New area = new length × new width

New area = 16 ft × 12 ft = 192 sq ft

Final Answer:

192
