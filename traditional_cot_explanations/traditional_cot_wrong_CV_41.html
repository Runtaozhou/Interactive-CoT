Problem Statement:

A child is making a pyramid by stacking some blocks. In the first row, he has 9 blocks. For the next rows, he has 2 less than the blocks he has from its bottom row. The pyramid has 5 levels. How many blocks did he use in all?

Step By Step Explanation:

Step 1: Calculate the number of blocks in the second row.

Since each row has 2 fewer blocks than the row below it, the second row will have:

First row blocks - 2 = 9 - 2 = 7 blocks

Step 2: Calculate the number of blocks in the third row.

Using the same pattern, the third row will have:

Second row blocks - 2 = 6 - 2 = 4 blocks

Step 3: Calculate the number of blocks in the fourth row.

Continuing the pattern:

Third row blocks - 2 = 4 - 2 = 2 blocks

Step 4: Calculate the number of blocks in the fifth row.

For the fifth and final row:

Fourth row blocks - 2 = 2 - 2 = 0 blocks

Step 5: Add up all blocks to find the total.

Total blocks = First row + Second row + Third row + Fourth row + Fifth row

Total blocks = 9 + 6 + 4 + 2 + 0 = 21 blocks

Final Answer:

21
