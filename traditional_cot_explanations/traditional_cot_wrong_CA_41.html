Problem Statement:

On a highway, the total number of vehicles passing is 300. If the number of cars on the highway is twice the number of trucks, find out the total number of trucks on the highway?

Step By Step Explanation:

Step 1: Define the number of trucks as a variable and express cars in terms of trucks.

Let's define t as the number of trucks.

Since the number of cars is twice the number of trucks, we can write:

cars + trucks = total vehicles

Substituting what we know: 2t + t = 300

Step 3: Simplify the equation to solve for the number of trucks.

We can combine like terms in our equation:

2t + t = 300

3t = 300

Step 4: Solve for the number of trucks.

To find t, we divide both sides of the equation by 3:
