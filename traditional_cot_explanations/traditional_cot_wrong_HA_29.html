Problem Statement:

Catriona has 8 goldfish in her aquarium. She has 4 more angelfish than goldfish. Also, she has twice as many guppies as angelfish. How many fish does Catriona have in her aquarium?

Step By Step Explanation:

Step 1: Calculate the number of angelfish.

We know that Catriona has 4 more angelfish than goldfish.

Number of angelfish = Number of goldfish + 4

Number of angelfish = 8 + 4 = 12 angelfish

Step 2: Calculate the number of betta fish.

We need to find the number of betta fish.

Number of betta fish = Number of goldfish + 3

Number of betta fish = 8 + 3 = 11 betta fish

Step 3: Calculate the number of guppies.

We know that Catriona has twice as many guppies as angelfish.

Number of guppies = Number of angelfish × 2

Number of guppies = 12 × 2 = 24 guppies

Step 4: Calculate the total number of fish.

We need to add all types of fish to find the total.

Total number of fish = Number of goldfish + Number of angelfish + Number of guppies + Number of betta fish

Total number of fish = 8 + 12 + 24 + 11 = 55 fish

Final Answer:

55
