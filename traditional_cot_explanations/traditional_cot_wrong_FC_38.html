Problem Statement:

Doris works at the Widget Factory in the packing department. She puts 3 widgets in each carton, which are 4 inches wide, 4 inches long, and 5 inches tall. She then packs those cartons into a shipping box before sending it to the loading bay. The shipping boxes are 20 inches wide, 20 inches long, and 20 inches high. How many widgets get shipped in each shipping box?

Step By Step Explanation:

Step 1: Calculate the volume of each carton.

Using the formula: width × length × height

Carton volume = 4 inches × 4 inches × 5 inches = 80 cubic inches

Step 2: Calculate the area of the shipping box.

Using the formula: width + length + height + width + length + height

Shipping box area = 20 + 20 + 20 + 20 + 20 + 20 = 120 square inches

Step 3: Find how many cartons fit in each shipping box.

Using the formula: shipping_box_area ÷ carton_volume

Cartons per box = 120 square inches ÷ 80 cubic inches = 1.5 cartons

Step 4: Calculate the total number of widgets in each shipping box.

Using the formula: widgets_per_carton × cartons_per_box

Widgets per box = 3 widgets per carton × 1.5 cartons = 4.5 widgets

Final Answer:

4.5
