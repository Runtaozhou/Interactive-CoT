Problem Statement:

In a national park, the number of redwoods is 20% more than the number of pines. If there are 600 pines in the national park, calculate the total number of pines and redwoods that are there.

Step By Step Explanation:

Step 1: Calculate how many less redwoods there are compared to pines.

We need to find the difference between redwoods and pines using the percentage:

Redwood difference = 20% × number of pines

Redwood difference = 20/100 × 600 = 120

Step 2: Calculate the number of redwoods in the park.

Using the difference we calculated:

Number of redwoods = pines - redwood difference

Number of redwoods = 600 - 120 = 480

Step 3: Calculate the total number of pines and redwoods.

We add the number of redwoods and pines to get the total:

Total trees = redwoods + pines

Total trees = 480 + 600 = 1080

Final Answer:

1080
